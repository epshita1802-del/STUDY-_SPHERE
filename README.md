# STUDY-_SPHERE
This project is a web-based study management system where students can access their studies semester-wise. Each student can navigate to their respective study pages through a structured dashboard. The system provides a study dashboard where students can view subjects, access learning materials, and revise topics easily.

Students can also view the complete syllabus for each semester, attempt tests to evaluate their understanding, and track their performance. The system maintains an overall progress report, allowing students to monitor their academic growth and identify areas for improvement.

The project is developed using HTML, CSS, and JavaScript for the front-end, and PHP with MySQL for the back-end, ensuring proper data management and smooth user interaction.
