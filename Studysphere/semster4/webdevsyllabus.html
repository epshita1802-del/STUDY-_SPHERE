Introduction to C++

C++ is a powerful general-purpose programming language developed by Bjarne Stroustrup.

It supports procedural, object-oriented and generic programming. It is widely used in software development, game development, and system programming.

Object Oriented Programming

OOP is a programming paradigm based on objects and classes.

Main concepts: Encapsulation, Abstraction, Inheritance, Polymorphism.
